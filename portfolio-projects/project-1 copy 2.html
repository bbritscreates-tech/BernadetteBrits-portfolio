Commerce Platform Redesign

A full overhaul of an outdated e-commerce platform to improve user flow, increase conversions and modernise the brand.

  • Role: UI/UX Designer, Developer
  • Timeline: 3 Weeks
  • Deliverables: Wireframes, UI, Branding, Development
Live Demo ‚Üí View on GitHub

Overview

The platform was outdated, slow and visually inconsistent. Users struggled to complete checkout, and drop-off rates were high. My role was to modernise, simplify and improve conversion.

Tools Used

Figma, Photoshop, Illustrator, VS Code, HTML/CSS/JS

Goals

  • Improve the visual identity
  • Streamline user flow
  • Enhance conversion rates
  • Modern UI that feels trustworthy and fast

The Problem

Users frequently abandoned the checkout process due to unclear navigation, long forms, and a visually cluttered interface. The brand also lacked a consistent design language.

The Solution

  • Completely redesigned checkout flow
  • Clear navigation and simplified visual hierarchy
  • Improved product detail layout
  • Modern, consistent design system
  • Added responsive design for mobile-first

Reflection

This project reinforced the importance of designing with data and simplifying the user journey. In a future iteration, I would perform usability testing earlier and explore motion design enhancements.
